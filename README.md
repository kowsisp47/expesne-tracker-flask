# Expense Tracker (Flask)

A simple expense tracker web application built using Flask.
Users can add products with price and delete them easily.

##  Features

* Add product with price
* Delete product
* Simple and clean user interface

## Technologies Used

* Python
* Flask
* HTML
* CSS

## How to Run

1. Install Flask:
pip install flask

2. Run the application:
python app.py

3. Open in browser:
http://127.0.0.1:5000

## Project Description

This project demonstrates add and delete operations using Flask.
